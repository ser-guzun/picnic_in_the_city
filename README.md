# Пикник в городе
Приложение для планирования пикника в городах. 

## Описание
Приложение позволяет планировать пикники в городе. 
В приложение имеются следующие сущности:
- город
- пользователь
- пикник

## Запуск
Приложение и БД запускаются в docker-контейнерах командой:
```bash
doker-compose up -d
```

## Старт
http://localhost:8000/
